2.2 The Logic: Calculating Velocity at the "Tag Cluster" Level
The Nano-Trend Velocity Index (NTVI) shifts the unit of analysis from the individual game to the Tag Cluster. Instead of tracking the performance of a single title, Rx Queen tracks the aggregate rank velocity of all games sharing a specific combination of AI-generated tags (e.g., "Subgenre: Screw Puzzle" + "Theme: Home Decor" + "Art Style: 3D Realistic").6
2.2.1 The Velocity Algorithm
To calculate the NTVI, we utilize the real-time rank volatility data across the monitored regions. The algorithm must account for the non-linearity of rank value; a move from Rank #1000 to Rank #500 is less significant than a move from Rank #50 to Rank #10.

$$V_{cluster} = \frac{\sum_{i=1}^{n} (w(R_{i,t}) \cdot \Delta R_{i}) \cdot S_{i}}{N_{cluster}}$$
Where:
â—	$V_{cluster}$: The Velocity Index for a specific tag cluster.
â—	$n$: The number of games within the tag cluster.
â—	$R_{i,t}$: The current rank of game $i$ at time $t$.
â—	$w(R)$: A logarithmic weighting function that assigns higher value to rank changes near the top of the charts (e.g., Top 50 > Top 200).
â—	$\Delta R_{i}$: The positive change in rank over a rolling 7-day window. Negative rank changes are dampened to focus on emergent trends.
â—	$S_{i}$: A "Signal Purity" coefficient. This uses the Ad Dependency Score (see Solution V) to down-weight games that are purely buying their rank via aggressive UA, thereby isolating organic viral interest.3
â—	$N_{cluster}$: The total count of active games in the cluster, normalizing for cluster size.
2.2.2 Thresholding and Lifecycle Tagging
The system applies statistical thresholding to classify trends into lifecycle stages:
1.	Emergent: High Velocity ($V_{cluster} > 2\sigma$), Low Game Count. Action: Greenlight Prototypes.
2.	Growth: High Velocity, Rising Game Count. Action: Fast Follow / LiveOps Pivot.
3.	Peak: Stable/Declining Velocity, High Game Count. Action: Avoid Entry / Focus on Differentiation.
4.	Saturn (Saturated): Negative Velocity, High Game Count. Action: Exit / Kill Prototypes.
This logic allows Rx Queen to detect, for example, that while "Puzzle" games are stable, the specific Screw Puzzle mechanic is in the "Emergent" phase in Brazil and "Growth" phase in the US, signaling an arbitrage opportunity.8
